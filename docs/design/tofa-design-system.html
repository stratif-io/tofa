07

Voice

How TOFA speaks

Sir Wink is a serious tool with a light personality. He's competent, never bumbling — but he doesn't lecture. Three principles:

01 · Brief

One screen, one line, one fact. If it can be a glance, don't make it a paragraph.

02 · Warm

Security tools are cold. We're not. Small jokes, occasional emoji, never sterile.

03 · Honest

Errors say what broke and how to fix it. Never blame the user. Never hand-wave.

Do
"Sir Wink's got it. 😉"
After successful add
Don't
"Operation completed successfully. Account credentials have been stored in the system keychain."
Sterile, twice the words, half the warmth
Do
"no account named 'notion' — did you mean notion-work?"
Error with a fix
Don't
"ERROR: AccountNotFoundException — please verify the identifier and try again."
Stack-trace energy, no help
Do
"2FA, in your tray."
Tagline: 4 words, says it all
Don't
"The premier macOS solution for time-based one-time password management."
